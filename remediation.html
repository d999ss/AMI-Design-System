Gap Inventory 12 Items

All identified gaps categorized by severity.

Incomplete Elevation System

Only "Elevation 2/Cast down" exists. Need Elevation 1 (subtle, cards), Elevation 3 (modals/dialogs), and Elevation 4 (tooltips/popovers).

3 Spacing Naming Systems

Space/8 vs Spacing/8 vs spacing/08. Components use different conventions.

2 Font Token Systems

Font/Font size/5 vs font size/05. PascalCase and lowercase coexist.

4 Font Families Active

Graphik (primary), Petersburg (secondary), F37 Lineca (mobile), Roboto (legacy). Roboto needs migration.

6+ Color Naming Systems

New semantic, Text & Icons, AMIO/Primary, /Primary, color/surface, Tertiary across 20+ components.

No Breakpoint Tokens

XL, L, M, S variants exist but no shared breakpoint variables.

No Grid Tokens

No column count, gutter width, or margin tokens. Only one layout token.

Status Color Inconsistencies

Error: #de3b2d vs #dc3426. Success: #0b8923 vs #065a18.

Missing Gradient Definitions

"Gradients/Light Marketing Banner" referenced but undefined.

Line Height Inconsistency

Ratio (1.2) vs px (24px, 30px) across headline styles.

Desktop/P3 Dual Definition

Same name for Graphik 13px (Cards) and Roboto 13px (Footer).

No Motion / Animation Tokens

No duration, easing, or transition tokens.

Prioritized Roadmap

P0 — Before Engineering Handoff

Items that block BTS from starting the build.

  • Resolve status color discrepancies — Pick canonical values for error red and success green
  • Define breakpoint tokens — Codify XL (1536), L (1279), M (768), S (480)
  • Document grid specifications — Column counts, gutters, margins per breakpoint
  • Complete elevation scale — Define Elevation 1, 3, 4 based on Elevation 2 pattern

P1 — Sprint 1: Token Standardization

Unify naming conventions and begin legacy migration.

  • Standardize color naming — Migrate to Color/{role}/{category}/{state}
  • Unify spacing tokens — Standardize on Space/{value}
  • Unify font tokens — PascalCase: Font/Font size/5
  • Migrate Footer from Roboto to Graphik

P2 — Sprint 2-3: Completeness

Fill remaining gaps and finish migrations.

  • Define gradient tokens — Fill "Gradients/Light Marketing Banner" and others
  • Add motion tokens — Duration (fast/normal/slow) and easing curves
  • Standardize line heights — Pick ratio vs px approach
  • Resolve Desktop/P3 dual definition

P3 — Ongoing: Refinement

Continuous improvement and maintenance.

  • Migrate remaining legacy components — Text & Icons, AMIO, color/surface, Tertiary
  • Source Frank's existing tokens — Merge into standardized structure
  • New work uses new system immediately — Legacy migration gradual
