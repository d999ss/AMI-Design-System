Gap Inventory 22 Items — Verified via Figma MCP

All identified gaps categorized by severity. Updated with findings from get_variable_defs extraction across all 29 documented components (Batch 1 + Batch 2).

3 Spacing Naming Systems Coexist

System 1: spacing/08 (lowercase — Button, Select, Tabs, Chip, NavBar, NavDrawer, Modal, Bottom Sheet). System 2: Space/8 (PascalCase — Text Input, Checkbox, Action Menu, Modal, Profile Menu). System 3: Spacing/8 (capitalized — Alert, ISI, Brand Container, Card, Progress Bar, Navigation XL). Modal and Profile Menu use BOTH spacing/ and Space/.

4 Font Token Systems Found

1: font size/05 (lowercase, 12 components). 2: Font/Font size/5 (PascalCase, 3 components). 3: Desktop/P3 (legacy composite — Footer 4.0, Card). 4: Body/Small, Caption/Regular (legacy composite — ISI, Card, Progress Bar). Footer 4.0 uses Roboto in Desktop/ tokens.

5 Corner Radius Naming Patterns

Corner/Small: 8, Corner/Extra small: 4, corner/extra_small: 4 (lowercase underscore), Corner/Circle: 999, Corner/Medium: 16 (Profile Menu only), Corner/Medium small: 12 (Text Input only).

4 Different Error Red Values

Button/Select: #de3b2d, Text Input/Checkbox: #dc3426, Badge text: #961307, Alert inpage: #ff0000. No canonical error color defined.

Line Height Inconsistency

Ratio (1.5 in Snackbar/Card Roboto, 1.2 in Headlines) vs px values (18/20/24/26/30px in semantic system).

AMIO/Primary — Oldest Token System

Progress Bar uses AMIO/Primary/Black and AMIO/Primary/Dusty Rose — the earliest naming convention found, pre-dating even Text & Icons/. Needs immediate migration.

Card: Fully Legacy (High Priority)

Product cards appear on every major screen. All 17 tokens use legacy naming: Text & Icons/, Desktop/P3, Color/Dusty Rose/100. Highest-impact migration target.

ISI: Fully Legacy System

Uses Text & Icons/, Color/White/100, Color/Neutral Grey/40. Contains unique large spacing values (80px, 124px) not found in the semantic system.

Footer 4.0: Pre-Rebrand Roboto

Uses /Primary / 000000 and Roboto font (not Graphik). Also references UI / Zoom Blue: #4188FB — a blue not used anywhere else in AMI.

Snackbar: Legacy /Primary + Roboto

Only 2 token bindings: /Primary / 000000 and Desktop/P2 (Roboto 16px).

Alert: Legacy Text & Icons + Color/Utility

Uses Text & Icons/Default, Color/Utility/Red (#ff0000 raw), and capitalized Spacing/ convention.

Navigation XL: Partial Migration Stalled

Contains both Text & Icons/ (legacy) and Color/Border/Neutral/Subtle 2 (new semantic). Migration was started but not completed. Also has Flowkit/White — a third-party tool artifact.

Toggle: Zero Token Bindings

All visual properties hardcoded. Must be fully tokenized before implementation.

Search: Zero Token Bindings

Core form control with no variable references. Should inherit Text Input field tokens (same family).

Brand Container: 4 Legacy Tokens Only

Brand-specific styling appears hardcoded per product. Only Spacing/16, Spacing/8, Color/Black/20, Color/Black/50. Needs brand token collections.

Footer (Latest): Only 2 Tokens

Fill / White and Icon/Neutral/White — among the least tokenized components. All other visual properties hardcoded.

Button: Old + New Tokens Coexist

Contains both Color/Container/Brand/Interactive/Enable (new) AND color/surface/interactive/Brand/Enable (old) pointing to the same values.

Elevation System Incomplete

"Elevation 2/Cast down" and "Elevation 2/Cast up" (Bottom Sheet) exist. Need Elevation 1 (cards), Elevation 3 (modals), Elevation 4 (high-priority tooltips).

Breakpoint Tokens Found but Informal

NavBar contains viewport/min: 1280, viewport/max: 2560, viewport/common: 1440. These exist but are not formalized across all responsive components.

No Grid Tokens

No column count, gutter width, or margin tokens. Only observable from screen designs.

Missing Gradient Definitions

"Gradients/Light Marketing Banner" referenced but undefined.

No Motion / Animation Tokens

No duration, easing, or transition tokens defined anywhere in the system.

Prioritized Roadmap

P0 — Before Engineering Handoff

Items that block BTS from starting the build.

  • Migrate Card tokens to semantic system — Cards appear on every screen. All 17 tokens use legacy naming (Text & Icons/, Desktop/P3, Color/Dusty Rose/).
  • Resolve status color discrepancies — Pick canonical error red from #de3b2d, #dc3426, #961307, #ff0000
  • Define breakpoint tokens — Formalize viewport/ tokens already in NavBar (min:1280, max:2560, common:1440)
  • Document grid specifications — Column counts, gutters, margins per breakpoint
  • Complete elevation scale — Elevation 2 exists (Cast down + Cast up). Need Elevation 1 (cards) and Elevation 3 (modals).

P1 — Sprint 1: Token Standardization + Legacy Migration

Unify naming conventions and migrate highest-impact legacy components.

  • Unify spacing tokens — Pick one of spacing/08, Space/8, or Spacing/8 across all 29 components
  • Unify font tokens — Standardize on one system (lowercase or PascalCase)
  • Migrate ISI to semantic system — Replace Text & Icons/, Color/White/100, Spacing/ with new equivalents
  • Migrate Footer + Snackbar from Roboto to Graphik — Two components still using pre-rebrand font
  • Migrate Progress Bar from AMIO/Primary/ — Oldest token system in the file
  • Tokenize Toggle and Search — Both have zero variable bindings
  • Remove Button legacy duplicates — Clean up color/surface/ tokens that duplicate new Color/Container/ tokens

P2 — Sprint 2-3: Completeness

Fill remaining gaps and finish migrations.

  • Define gradient tokens — Fill "Gradients/Light Marketing Banner" and others
  • Add motion tokens — Duration (fast/normal/slow) and easing curves
  • Standardize line heights — Pick ratio vs px approach
  • Resolve Desktop/P3 dual definition

P3 — Ongoing: Refinement

Continuous improvement and maintenance.

  • Migrate remaining legacy components — Text & Icons, AMIO, color/surface, Tertiary
  • Source Frank's existing tokens — Merge into standardized structure
  • New work uses new system immediately — Legacy migration gradual
