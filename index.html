Gaps & Issues

Critical items to address for system completeness

No Elevation / Shadow Tokens

No shadow or elevation variables defined anywhere in the file. Needed for cards, modals, dropdowns, tooltips.

Incomplete Radius Scale

Only Small (8), Medium Small (12), and Circle (999). Missing Medium, Large, XL values for containers and cards.

Inconsistent Spacing Names

Three conventions: Space/8, spacing/08, Spacing/12. Standardize on one.

Inconsistent Typography Tokens

New components use Font/Font size/5, older ones use font size/05. Case and format differ.

Two Font Families

New system uses Graphik. Legacy Footer uses Roboto with completely different style naming (Desktop/P1, Desktop/P3).

Three Color Systems

New semantic (Color/Container/...), legacy (/Primary / 000000), and old nav (Color/White/100) all coexist.

No Breakpoint Tokens

No responsive breakpoint variables defined. Needed for Medium and Large viewport consistency.

Legacy Token Systems

Components using outdated naming conventions that need migration

Recommendation

  • Standardize on the new semantic naming: Color/Container/..., Space/..., Corner/...
  • Build out missing tokens: elevation/shadows, full radius scale, breakpoints
  • Normalize spacing token names to Space/{value}
  • Normalize typography tokens to Font/{category}/{value}
  • New work follows new token structure immediately
  • Migrate legacy components (Footer, Side Nav) gradually over coming weeks
  • Source Frank's existing tokens and merge into standardized structure
