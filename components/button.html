Button

Primary interactive element for triggering actions. The most variant-rich component in the system with 308+ configurations. Uses 3 sizes, 4+ color schemes (Neutral, Brand, Mute, Destructive), 5 states, solid/ghost/outline styles. Both new semantic and legacy color/surface tokens coexist on this component.
