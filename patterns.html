Interaction Patterns

Observable state patterns for interactive components documented from the Figma component library.

Modals & Dialogs

Open/closed states with backdrop overlay. Size variants for small confirmations and large content modals. Alert modal variant for destructive action confirmation.

Components: Modal, Alert Modal, Quick Start Pop-up

Dropdowns & Select

Open/closed toggle with elevation shadow. Single-select and multi-select with checkmarks. Searchable variant with type-ahead filtering. 72 total configurations.

Components: Select, Action Menu

Accordion Expand/Collapse

Chevron rotation on expand/collapse. Single-expand and multi-expand modes. Used for FAQs, filter panels, and progressive disclosure.

Components: Accordion, Filter Panel

Tab Navigation

Horizontal tab bar with active underline indicator in brand color. Content swap on tab selection. Badge counters on tabs for unread/count states.

Components: Tabs

Tooltips & Popovers

Four position variants (top, bottom, left, right) with directional arrow. Dark background with white text. Triggered on hover/focus of target element.

Components: Tooltip

Navigation Drawer

Expanded/collapsed side navigation with icon-only collapsed state. Active item highlighting with brand color. Used on desktop widths, converts to hamburger menu on mobile.

Components: Consumer NavBar, Side Nav

Snackbar Notifications

Temporary bottom-of-screen notification. Auto-dismiss after timeout. Optional action button and manual close. Error variant in red.

Components: Snackbar

Form Validation

Real-time validation on blur/submit. Error state with red border + error message below input. Success state with green checkmark. Disabled state with reduced opacity.

Components: Text Input, Select, Checkbox
